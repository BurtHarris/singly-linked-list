# singly-linked-list

Simple class to build a 'singly linked list'

```
// Given a sequence [1, 2, 3]
const s = new SinglyLinkedList([1, 2, 3]);
s.print() // [{ value: 1, next: 2 }, { value: 2, next: 3 }, { value: 3 }]
```

## Getting started
`npm install`

## Run SinglyLinkedList
`npm run start`

## Test
`npm run test`
